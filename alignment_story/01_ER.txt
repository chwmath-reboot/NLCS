이 이류무인의 성장을 돕기 위한 시뮬레이션 게임을 만들거야.

시작은 베테랑 간호사 선생님의 호출로 시작할거야.

간호사지만 실제 절정 무인의 감을 가지고 있어서, 인턴에게 넘기기 좋을 환자를 본능적으로 골라내지.

"김선생님, 이 환자 좀 봐주세요."

대화창과 차트창을 분리해서 작업을 해줘.

### 환자를 먼저 생성하고, 생성된 조합을 기준으로 환자가 대답을 하면 돼.
환자는 10명당 9명은 감기를 기준으로 증상을 조합해서 생성해주면 돼.
증상에 적당한 체온을 같이 생성하는 걸 잊지마.
10명당 1명은 부비동염을 기준으로 증상을 조합해서 생성해주면 돼.
단, 부비동염 환자의 기본 대사는 감기 환자의 기본 대사를 이용하면 돼.
>> 체온은 발열이나 오한 증상이 선택된 경우 일반적인 범위에서 높게 제시
>> 발열이나 오한 증상이 없는 경우 일반적인 범위에서 낮게 제시
>> 부비동염은 부비동염의 체온범위를 기준으로 증상을 만들면 되는데, 고온이 나는 부비동염 환자의 증상을 만들때 자료를 같이 검토해보자.

차트에 환자의 증상을 기록하는 형태로 진행이 되면 돼.

우선 인턴인 김선생님이 질문을 해.

"환자분 어디가 불편하세요?"

그리고 환자의 대답을 듣고, 인턴이 증상 버튼을 눌러서 증상을 기록을 하면 돼.
>> 환자는 자신의 증상을 한번에 대답해도 되고, 일부만 이야기 해도 돼. 다만 2차 질문에선 빠짐 없는 대답을 해야해. 부부동염 관련 증상은 따로 말하지 않을 거야.

환자의 1차 대답 출력

1차 대답에 따른 증상을 모두 클릭하면, 인턴의 질문을 출력

"추가로 더 불편한 곳은 없으세요?"

환자의 2차 대답 생성
>> 1차에 증상을 다 말한 경우는 "다른 곳은 괜찮아요."를 출력하면 돼.

베테랑 간호사가 체온을 알려준다.
"환자분 체온은 XX.X도 예요."

>> 체온은 차트에 자동 기록해준다.

[환자의 응답은 감기 증상의 환자 대사에서 랜덤으로 골라줘.]

증상은 클릭창 형태로 제시를 해줘.

예시) 콧물, 인후통, 마른 기침, 가래 기침 등 환자의 대사 옆에 붙은 설명을 버튼으로 일괄 제시해줘. 

버튼의 작동은 게임 화면에 클릭을 반영해서 적어주면 돼.

✅ <함정 증상 – 부비동염(Sinusitis)>

(감기와 거의 동일하게 보이지만 특정 포인트에서 상급자 보고 필요)

============================
■ 개요

부비동염은 초기엔 감기와 구분이 거의 불가능하지만,
특정 질문에 걸리면 상급자 호출 루트가 열리는 구조.

감기와 증상이 90% 동일

“조금 오래간다 / 특정 통증 위치가 있다 / 콧물의 성상 변화” 등
미묘한 힌트에서 진짜 정답이 갈림

인턴 플레이어에게 문진 능력 테스트가 되는 함정 환자

============================
■ 1) 감기와 동일하게 보이는 증상들
============================

부비동염 환자에게도 흔히 나타나는 감기류 증상:
코막힘
콧물
기침
가벼운 발열
피로감
두통
→ 표면상 완전 감기처럼 보임.
→ 인턴이 “아 그냥 감기네”라고 판단하기 딱 좋다.

✔ 1) 대부분 미열(37.5~38.0°C대)
감기와 비슷한 수준의 미열이 흔함
그래서 인턴 입장에서 감기와 구분하기 매우 어렵다

✔ 2) 심한 부비동염 = 열이 더 높아질 수 있음
염증이 부비동(코 주변 공간)에 쌓일수록
38.0~38.5°C 정도로 올라갈 수 있음
특히 세균성 부비동염이면 열이 더 뚜렷해짐

✔ 3) 열이 없을 수도 있음
어떤 환자는 “딱히 열은 없는데 두통·안면통만 심함”
이게 오히려 부비동염의 함정 포인트

✔ 4) 열의 패턴이 감기와 거의 동일
→ 발열만으로는 감기 vs 부비동염 구분 불가능
→ 반드시 특이 증상 A~D에서 판단해야 함

============================
■ 2) 상급자 보고가 필요한 부비동염 특이 증상
============================

아래 특이 증상 1개라도 존재하면 → 상부 보고 루트

모두 부정 → 감기 루트

나는 네 질문 구조 그대로 쓰면서 대사 2~3개씩 확장해줄게.

✔ 특이 증상 A — 심한 안면 통증(얼굴 압통)

볼, 눈 아래(광대 아래), 이마 등을 누르면 찌릿하거나 욱신거림

“얼굴이 막힌 느낌” “찌르는 통증” 같이 묘사됨

인턴 질문

"혹시 얼굴이나 눈 주변이 아프거나 눌렀을 때 더 아프진 않으세요?"

환자 대사

“네… 얼굴이 꽉 막힌 느낌이 들어요.” >> [상부 보고]

“눈 아래가 계속 욱신거려요.” >> [상부 보고]

“여기를 누르면 찌릿하게 아파요.” >> [상부 보고]

“얼굴 전체가 무겁게 쑤시는 느낌이에요.” >> [상부 보고]

✔ 특이 증상 B — 진득하고 악취 나는 콧물(색 변화)

감기 = 묽고 투명

부비동염 = 노랗거나 초록빛, 진득함, 악취

인턴 질문

"콧물이 진하고, 색이 변했거나 냄새가 강하게 나진 않으세요?"

환자 대사

“코를 풀면 휴지에서 냄새가 강하게 느껴져요.” >> [상부 보고]

“노란색? 초록색? 좀 진하더라고요.” >> [상부 보고]

“콧물이 끈적거려서 잘 안 나와요.” >> [상부 보고]

“예전 감기 때보다 콧물 색이 훨씬 진해요.” >> [상부 보고]

✔ 특이 증상 C — 지속적 기침(특히 밤에 심해짐)

감기 기침보다 훨씬 오래 지속

밤에 누우면 심해지는 경향

기간이 길어지는 것이 포인트

인턴 질문

"기침은 언제부터 하셨어요? 그리고 밤에 더 심해지시나요?"

환자 대사

“어제부터 시작된 것 같아요.” >> [감기]

“한 사나흘 정도인데, 밤에 특히 심해요.” >> [상부 보고]

“일주일 넘었는데 계속 기침이 안 가라앉아요.” >> [상부 보고]

“밤에 누우면 기침이 몰아서 나요.” >> [상부 보고]

✔ 특이 증상 D — 몸을 숙일 때 악화되는 두통

세수·양치할 때 고개 숙이면 통증이 확 증가

압력 증가 시 악화되는 진양성 두통

인턴 질문

"두통이 고개 숙일 때 더 심해지지는 않으세요?"

환자 대사

“아니요, 특별히 달라지진 않아요.” >> [감기]

“어! 맞아요. 세수하기가 힘들 정도예요.” >> [상부 보고]

“숙이면 눈 뒤쪽이 찌릿하게 아파요.” >> [상부 보고]

“고개를 숙이면 머리가 무거워지면서 통증이 확 올라요.” >> [상부 보고]
